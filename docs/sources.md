# 対応データソース詳細

PaperFetchがサポートする、またはサポート予定のデータソースについての詳細です。

## 1. Arxiv (主要ソース)
- **概要**: 物理学、数学、計算機科学などのプレプリントサーバー。
- **機能**: APIを使用した高速かつ安定した検索。
- **取得内容**: PDF、メタデータ（タイトル、著者、要約）。

## 2. IEEE Xplore
- **概要**: 電気工学、コンピュータ科学分野の主要なデータベース。
- **制限事項**: APIキーを使用しないスクレイピング/Open Accessフィルタリングによる取得のため、アクセス権限のない有料論文はダウンロードできません。
- **取得内容**: Open Access論文のPDF。

## 3. 3GPP
- **概要**: 移動通信システムの標準化団体による仕様書や寄書。
- **検索方式**: Google Custom Search を利用したドキュメント検索。
- **特長**:
    - **URL指定ダウンロード**: FTPサーバー等のURLを直接クエリとして入力することで、そのディレクトリ配下のファイルを一括ダウンロードする機能があります。
    - **自動変換**: `.doc` / `.docx` 形式で配布されている仕様書を、ダウンロード後にLibreOffice（要別途インストール）を用いてPDFに変換するオプションがあります。

## 4. Google Patents (USPTO) [Experimental]
- **概要**: 米国特許商標庁の特許情報。
- **Status**: ⚠️ **実験的機能 / 未検証**
- **注意**: PatentsView APIを利用する実装が含まれていますが、現在はAPIキーの制約等により完全な動作検証が行われていません。利用は推奨されません。
